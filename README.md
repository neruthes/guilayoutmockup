# GUILayoutMockup

Tool for making GUI layout mockup.




## Installation

### CSS in head

```html
<link rel="stylesheet" href="src/main.css" />
```

### JS at the end of body

```html
<script src="src/main.js"></script>
```


## Example

```html
<div id="MegaContainer">
    <div class="Splash">
        <div class="Page">
            Page UI content here
        </div>
        <div class="Notes">
            Extra notes here
        </div>
    </div>
</div>
```
